Why, How, What Analysis
WHY - Problem Statement
Core Challenge: Traditional pedigree relationships give expected values, but actual genomic sharing varies substantially due to Mendelian sampling and linkage.
  • Existing formulae were scattered and incomplete for different relationship types
  • No unified framework existed for computing variances across all relationship types
  • Genomic selection and GWAS require understanding of actual vs. expected relationships
  • Need to quantify uncertainty in relationship inference from genomic data
Applications: Critical for genomic selection, relationship inference, GWAS quality control, and estimating inbreeding depression.
HOW - Mathematical Framework
Unified Approach: Systematic mathematical framework using identity-by-descent (IBD) indicators and recombination mapping functions.
  • IBD Indicators: k‚ÇÄ, k‚ÇÅ, k‚ÇÇ for sharing 0, 1, or 2 allele pairs
  • Mapping Functions: Haldane's function to relate physical and genetic distances
  • Integration: Chromosome-level integration over recombination fractions
  • Generalization: Extensions for sex-linked inheritance, different map lengths, and complex pedigrees
Innovation: First comprehensive treatment covering all major relationship types with computational formulae.
WHAT - Key Findings
Major Results: Comprehensive variance formulae and surprising insights about relationship uncertainty.
  • High coefficients of variation for distant relationships (CV > 1 for cousins)
  • Substantial skewness increases with relationship distance
  • Different pedigrees with same expected relationship have different variances
  • Practical implications for genomic selection and relationship inference
Impact: Foundational theory enabling modern genomic applications and relationship studies.

Software Implementation
R Code Structure: Efficient computation for genetic applications 
# Core w_n function
w_function <- function(n, l) {
    if (n == 0) return(0)
    
    sum_term <- 0
    for (r in 1:n) {
        coeff <- choose(n, r)
        term <- (2*r*l - 1 + exp(-2*r*l)) / (r^2)
        sum_term <- sum_term + coeff * term
    }
    
    return((1/(2*l^2)) * (1/4)^n * sum_term)
}

# Half-sib variance example
var_half_sib <- function(l) {
    return(4*w_function(2, l) - 2*w_function(1, l))
}

Inbreeding Applications
Actual vs. Pedigree Inbreeding
Key Insight: Variation in actual inbreeding directly parallels relationship variation
Inbreeding-Relationship Connection
F(offspring) = Œ∏(parents)
Var[F(offspring)] = Var[Œ∏(parents)]
Actual inbreeding follows genomic coancestry of parents
Practical Example:
  • Cousin mating: Expected F = 0.0625
  • Standard deviation = 0.024
  • Coefficient of variation = 38.4%
Variance Partitioning
Between vs. Within Family Components: Decomposing inbreeding variance
Variance Decomposition
Var(Ftotal) = Var(Fbetween) + Var(Fwithin)
Var(Fbetween) = ¬ºVar(Rparents)
Full Sib Mating Example:
  • Total variance: 0.00191
  • Between families: 0.00038 (20%)
  • Within families: 0.00152 (80%)

Implication: Most inbreeding variance occurs within families, making family-based studies powerful for detecting inbreeding effects.

Inbreeding Depression Studies
Enhanced Methodology: Using actual rather than pedigree inbreeding
  • Traditional approach: Regress phenotype on pedigree F
  • Genomic approach: Regress phenotype on actual F
  • Advantage: Within-family analysis eliminates environmental confounding
  • Power: Higher precision with dense markers
Best Practice: Combine pedigree information with genomic data to maximize power for detecting inbreeding effects on fitness traits.

Limitations and Future Directions
Current Limitations
Theoretical Assumptions:
  • Haldane mapping: Assumes no interference, may overestimate recombination
  • No structural variation: Doesn't account for CNVs, inversions
  • Infinite sites: Assumes sufficient marker density
  • Non-inbred founders: Extensions needed for inbred populations
Computational Challenges:
  • Complex pedigrees: Limited formulae for irregular structures
  • Population structure: Admixture complicates relationship inference
  • Rare variants: Low-frequency alleles affect IBD detection
Future Research Directions

Methodological Extensions: Incorporate modern genomic complexities including structural variants, population stratification, and admixture. Develop efficient algorithms for complex pedigrees and integrate with sequence-based approaches.

Applications: Extend to conservation genetics, personalized medicine, and evolutionary studies. Combine with functional genomics to understand the biological significance of relationship variation.
