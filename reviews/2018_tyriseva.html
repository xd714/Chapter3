Research Context & Problem Statement
Critical Gap in International Evaluation: The multiple-trait across-country evaluation (MACE) method is highly sensitive to changes in genetic variance over time in national evaluations. Bulls from birth year classes with inflated genetic variances receive upward bias in ranking, but no validation methods existed to test for homogeneity of genetic variance across years.

Detailed Methodology Flow
1
Genetic Variance Estimation Framework
Develop method based on Mendelian sampling (MS) and prediction error variance (PEV)
Core Equation: Within-year genetic variance using MS
Key Innovation: Use approximated reliabilities instead of exact PEV calculation
Reliability Threshold: MS reliability > 0.1 for numerical stability
2
Statistical Testing Framework
Implement weighted linear regression and outlier detection
Trend Test: Weighted linear regression of genetic variance on year
Bootstrap: 1,000 samples for 95% confidence intervals
Outlier Test: Detect years that don't fit linear trend with Bonferroni correction
3
Simulation Study Design
Comprehensive validation using realistic population structures
Data Sources: Danish Holstein protein yield test-day records
Scenarios: Control (homogeneous), T2 (2% yearly increase), T10 (10% yearly increase)
Population Sizes: Moderate (M: 756,537 cows, 290 bulls/year) and Small (S: 483,410 cows, 27 bulls/year)
4
Data Structure and Editing
Establish data requirements and quality control measures
Time Period: 22 years (1988-2009) for comprehensive trend detection
Data Edit 1: EBV reliability ‚â• 0.50 (removed edge effects but caused selection bias)
Data Edit 2: MS reliability > 0.1 (final criterion, removed 26% cow records, 0.3% bull records)

Comprehensive Results Analysis
WHY - International Quality Control Need
MACE Sensitivity: International dairy bull evaluations are biased by heterogeneity of genetic variance in national evaluations.
  • Bulls from countries with inflated variance trends gain unfair advantage
  • Scaling effects from increasing production levels over time
  • Combined evaluations from different management systems
  • No existing validation methods for variance homogeneity
Impact: Unreliable international rankings affecting global breeding decisions.
HOW - Validation Method Development
Core Innovation: Practical method for routine validation of genetic variance homogeneity.
  • Estimation: Use MS and approximated PEV for within-year genetic variances
  • Testing: Weighted linear regression with bootstrap confidence intervals
  • Validation: Compare against gold-standard FMS method
  • Implementation: Account for inbreeding effects and reliability thresholds
Key Requirements: MS reliability > 0.1, tolerance values for large datasets.
WHAT - Performance Validation Results
Accuracy: Close agreement with reference method and simulated values.
  • MSVT: 245.4 kg¬≤ vs FMS: 252.7 kg¬≤ (simulated: 247 kg¬≤)
  • Trend Detection: 100% sensitivity for moderate populations with generated trends
  • Specificity: Acceptable false-positive rates with tolerance values
  • Small Populations: Reduced sensitivity but still functional
Practical Impact: Enables routine validation for international evaluation quality assurance.

Technical Specifications & Requirements
Data Requirements
Minimum Standards:
  • MS Reliability > 0.1: Essential for numerical stability
  • Birth Year Classes: At least half of average class size
  • Time Period: Sufficient years for trend detection
  • Low Heritability: Use bull data only (cows need ‚â•3 records)
Recommended Edits:
  • Exclude animals with incomplete pedigree information
  • Remove last birth year if class size < 50% of average
  • Apply appropriate tolerance values for large datasets
Implementation Guidelines
Tolerance Values:
  • Trend Tolerance: 2-4% yearly increase may be acceptable
  • Outlier Tolerance: ¬±10% of average genetic variance
  • Large Datasets: Statistical power requires practical thresholds
Inbreeding Effects:
  • Account for inbreeding in both EBV prediction and variance estimation
  • Ignoring inbreeding leads to slight negative trends
  • Effect is tolerable but ideally should be modeled
Method Comparison Summary
Sensitivity Analysis
MSVT Strengths:
  • Computationally feasible for routine use
  • Good accuracy when reliability requirements met
  • Applicable across different evaluation models
  • Suitable for international validation framework
MSVT Limitations:
  • Sensitive to approximated reliability quality
  • Less robust than FMS for low reliability data
  • Requires careful data editing
  • Reduced performance in small populations
Practical Recommendations
For Large Populations:
  • Apply tolerance values to avoid detecting trivial trends
  • Use both cow and bull data with appropriate filtering
  • Set practical thresholds for trend significance
  • Regular validation as part of quality control
For Small Populations:
  • Focus on bull data for better power
  • Accept reduced sensitivity for moderate trends
  • Emphasize detection of large variance changes
  • Consider longer time periods for trend detection
